# Failure Buckets

Choose one primary bucket when local gain does not translate.

- `adapter_missing`
- `circuit_mismatch`
- `objective_mismatch`
- `foundation_mismatch`
- `numerical_instability`
- `data_regime_mismatch`
- `mechanism_illusion`
- `implementation_bug`

If several are plausible, rank them. Do not average them into a vague conclusion.
