# Research Platform Ops Adaptation Guide

When adapting this skill to a project, bind these abstract roles:

- repo snapshot:
  branch, commit, worktree, dirty-state source
- execution lineage:
  run manifests, run ledgers, command logs, heartbeat logs
- evaluation artifacts:
  comparison reports, claim reports, paired deltas, benchmark tables
- long-form note:
  where the project stores experiment reflections
- durable claim memory:
  append-only claim ledger plus current hypothesis view

If a project does not yet have all five layers, add the missing layers before expecting cumulative research behavior.
